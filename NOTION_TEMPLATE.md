## 📅 行程安排

### 🏙️ 重庆部分（3月12-15日）

#### 到达当天（3月12日）
- [ ] 机场到酒店
- [ ] 在酒店附近（解放碑商圈）逛吃

#### Day 1（3月13日）
- [ ] **上午**：佳实养老院 – 从上往下逛
- [ ] **中午**：山城步道（10分钟）→ 十八梯（午餐）
- [ ] **下午**：大隧道遗址 → 博物馆
- [ ] **下午晚些**：八一好吃街（5分钟）
- [ ] **傍晚**：解放碑（2分钟）
- [ ] **晚上**：莱佛士 – river → 洪崖洞（10分钟 或 地铁6号线 + 江滩公园）

#### Day 2（3月14日）
- [ ] **上午**：地铁2号线 → 李子坝轻轨穿楼（重庆8D魔幻）→ 人民大礼堂 → 三峡博物馆
- [ ] **下午**：观音桥美食街（晚餐）→ 北仓文创街 → 重庆大融城 → 北城天街
- [ ] **晚上**：重庆无人机表演 ⭐ 20:30（周六限定）
  - 观看地点：长嘉汇购物公园观景台（4层或6层，有座位）
  - 表演时长：18分钟（3分钟灯光仪式 + 15分钟无人机秀）
  - 亮点：5,000+无人机同步表演，3D立体图案
- [ ] **晚上**：星光68广场B区 3层楼高的水族馆（周末3PM/7PM美人鱼show）
- [ ] **深夜**：九街（夜生活 NightLife）
